# iOS App Setup

## Quick Start (Xcode)
1. Xcode > File > New > Project > iOS App
2. Product Name: FamilyOutings
3. Interface: SwiftUI, Language: Swift
4. Create project under this folder: `/Users/ittonagai/Documents/family-outings/ios`
5. Replace the generated `FamilyOutingsApp.swift` with this file:
   - `/Users/ittonagai/Documents/family-outings/ios/FamilyOutingsApp.swift`
